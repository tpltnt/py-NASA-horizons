NASAHorizons overview¶

A pure python 3 wrapper for the NASA HORIZONS data service.

class NASAhorizons.NASAhorizons[source]¶

A python wrapper for the NASA HORIZONS data service telnet interface to query xyz-coordinates of objects. It uses the ICRF/J2000.0 reference frame.

convert_NASA_to_ISO_datestring(nasadate)[source]¶

Convert a NASA string to an ISO 8601 like format. The time is in Barycentric Dynamical Time.

Parameters:nasadate (str) – NASA date string
Returns:str in ISO-like format
Raises :TypeError 
>>> convert_NASA_to_ISO_datestring("A.D. 1977-Sep-10 00:00:00.0000")
1977-09-10T00:00:00.0000
>>> convert_NASA_to_ISO_datestring("B.C. 1977-Sep-10 23:10:00.0000")

get_data(start, end, format='list')[source]¶

Retrieve data (xyz-coordinates) for formerly selected object. A session will be initialized in the background if needed.

Parameters:
  • start (datetime.date) – date of first datapoint to be requested
  • end (datetime.date) – date of last datapoint to be requested
  • format (str – “list” (of dictionaires (default)) or “json“) – format of returned data
Returns:

data in selected format
Raises :

TypeError

 
>>> nasa = NASAhorizons()
>>> nasa.set_object_id(199)
>>> start = datetime.date(year=1977, month=9, day=10)
>>> end = datetime.date(year=1977, month=9, day=12)
>>> nasa.get_data(start, end, format="list")
[{'z': -0.02967482538878673, 'y': 0.02365967857453419, 'x': 0.3497488933057855, 'date': '1977-09-10T00:00:00.0000'}, {'z': -0.02647449350730847, 'y': 0.05283430192085586, 'x': 0.3408485768468899, 'date': '1977-09-11T00:00:00.0000'}, {'z': -0.02308289285966815, 'y': 0.08159238988582097, 'x': 0.3294953295232826, 'date': '1977-09-12T00:00:00.0000'}]
