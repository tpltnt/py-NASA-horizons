NASAHorizons overview¶

A python 3 wrapper/interface to the NASA HORIZONS data service.

class NASAhorizons.NASAhorizons[source]¶

A python wrapper for the NASA HORIZONS data service telnet interface to query xyz-coordinates of objects. It uses the ICRF/J2000.0 reference frame.

close_session()[source]¶

Explictly close the session.

convert_NASA_to_ISO_datestring(nasadate)[source]¶

Convert a NASA string to an ISO 8601 like format. The time is in Barycentric Dynamical Time.

convert_to_NASA_date(dateobject)[source]¶

convert given datetime.date-object to string in NASA format, e.g. 1977-Sep-10

create_session()[source]¶

Creates a new telnet connection to the NASA HORIZONS data service. Calling this method will destroy your old connection.

get_data(start, end)[source]¶

Retrieve data (xyz-coordinates) for formerly set object.

arguments: * start: datetime.date indicating first datapoint to be requested * end: datetime.date indicating last datapoint to be requested

has_session()[source]¶

A simple self test if a session already exits.

>>> nasa = NASAhorizons()
>>> nasa.create_session()
>>> nasa.has_session()
True
set_object_id(idnumber)[source]¶

Select object by its HORIZON internal ID number. Note that these are not the IAU or designation numbers. Negative values indicate spacecrafts.
